Should Be Equal ใช้เปรียบเทียบค่า มักจะมาพร้อมกับ msg คือการกำหนด “ข้อความ Error” 
| Keyword                     | ใช้กับ                     |
| --------------------------- | -------------------------- |
| Should Be Equal             | เปรียบเทียบทั่วไป          |
| Should Be Equal As Strings  | เปรียบเทียบข้อความ         |
| Should Be Equal As Integers | เปรียบเทียบตัวเลขจำนวนเต็ม |
| Should Be Equal As Numbers  | เปรียบเทียบตัวเลขทศนิยม    |
-------------------------- -------------------------- 
 EX     Should Be Equal As Integers    ${count}    6
    ...    msg=คาดว่าจะมีสินค้า 6 รายการ แต่พบ ${count} รายการ  
-------------------------- -------------------------- 
อธิบาย EX ตรวจสอบว่าค่าตัวเลขกำหนดคือ 6 ค่าเท่ากันหรือไม่ 
-------------------------- --------------------------
